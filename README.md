# Flappy Bird Clone 
A simple Flappy Bird clone built in Java, focusing on core game logic and basic collision detection.

# Features
- Keyboard-controlled bird movement
- Continuous obstacle generation
- Collision detection with pipes and boundaries
- Score tracking based on survival time
- Restartable game loop

# Tech Stack
- **Language:** Java  
- **GUI Framework:** Java Swing / AWT   

## How to Run the Project

### Prerequisites
- Java JDK 8 or above
- Git installed
   
### Steps
1. Clone the repository:
   - git clone https://github.com/DefinitelyNotHB/FlappyBird-Clone.git
2. Navigate to project directory
   - cd FlappyBird-Clone/flappyBird/src
3. Compile the source file
   - javac App.java
4. Run the game
   - java App
